Crumb & Co. is a bakery management website created for managing daily bakery operations. The website uses a soft beige and cream theme to match a cute and cozy bakery style. It mainly focuses on pastries, while still including a small selection of cakes.

Main Features

- Login page with simple username and password validation
- Dashboard page showing total sales, orders, menu items, best seller, real-time clock, weekly sales chart, bestseller chart, and recent orders
- Pastries page displaying pastry and cake products, prices, categories, and availability
- Orders page for tracking customer orders, quantity, total price, pickup time, and order status
- Reports page showing monthly revenue, customers, items sold, top product, sales trend, sales by category, and performance summary
- Logout confirmation popup for better user interaction
- Responsive and aesthetic design using HTML, CSS, JavaScript, Bootstrap, and Google Fonts
